Om te beginnen, moet je het HTML-bestand laten weten dat je CSS wilt gebruiken. Dit doe je op twee manieren. Of je zegt het volgende in je bestand: 

<style>Hier kan je de CSS in stoppen, die je hier leert in deze cursus</style>
Ook kan je dit in de head zetten: 
<link>rel="stylesheet" href="style.css">



Vervolgens kan je in deze stylesheets de data van objecten en stijlen geven. Dit kan je als volgt doen:

<style>
      body {
        color: red;
        background-color: blue;
      }
      </style>

Je moet eerst het object geven wat je wilt stijlen, in dit geval de body. Dit is dus de hele pagina. Zo zou je nu tekst in het rood krijgen en de achtergrondkleur in het blauw. Ook kan je een div stijlen. Dit doe je met #container of .container. Een # is id, een . is class. Hieronder zie je een aantal voorbeelden met dingen die je zou kunnen stijlen en op welke manier:


<style>
    #textcontainer {  /* Dit is een tekstvak */
      color: red;
      font-family: Arial, Helvetica, sans-serif;
      font-size: 18;
      background-color: blue;
    }

    button {
      height: 50px;
      background-color: lightblue;
      border-radius: 25px; /* Dit is hoe rond de knop is*/
      border-style: solid; /* Dit zorgt ervoor dat de border er hetzelfde uitziet, en niet met rare lijnen*/
      border-width: 5px;
    }
    </style>

Dit zijn gewoon wat voorbeelden hoe je een tekstvak en knop zou kunnen stijlen.

Als je denkt er klaar voor te zijn, kan je met onderstaande knop de oefening starten. Je zult onderweg vragen beantwoorden, en je kan je voortgang volgen op de pagina Mijn Overzicht.
