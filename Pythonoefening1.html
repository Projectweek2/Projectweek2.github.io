Python


Welkom bij de cursus van Python! Python is een taal om de computer te vertellen wat 'ie moet doen. Verschillende programma's voor Windows zijn hiermee gemaakt, en het is ook een belangrijke taal voor AI. Het is gemaakt door Guido van Rossum, een Nederlander! In deze cursus kan je verschillende aspecten van Python leren!


Om te beginnen, moet je Python hebben op je computer. Dit kan je doen op python.org en vervolgens heb je, net zoals met HTML, CSS en JS, een code-editor nodig. Een populaire keuze is Visual Studio Code. Vervolgens kan je beginnen met het maken van je code!
