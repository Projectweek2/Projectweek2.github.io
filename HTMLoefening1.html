HTML

Uitleg 1 en Oefening 1


Welkom bij de cursus van HTML! Vanaf hier start de reis naar het maken van je eigen websites! We houden het zo kort en krachtig mogelijk, met veel oefeningen zodat je het goed in de praktijk kunt brengen!


Een website bestaat uit een standaard 'format'. Zo moet een website voor de beste weergave altijd de volgende tags hebben:

<html>Dit is de data voor de browser zodat het kan zien dat het een HTML-bestand is</html>

<head>Hier kan je gegevens zoals icoon, titel en zoekwoorden voor een zoekmachine inzetten</head>

<body>Hier staat de inhoud, zoals tekst, afbeeldingen en linkjes</body>


In HTML moet je altijd pijltjes gebruiken (zie voorbeelden hierboven). Een aantal bekende tags zijn:


<a href="linknaarwebsite.html">Dit is de tekst waar je op kan klikken</a>

<H1>Dit is koptekst</H1>

<img src="LinkNaarAfbeelding.png">



<!--Dit is een opmerking zodat je kan zien wat de code doet-->

Als je er klaar voor bent, kan je met onderstaande knop de oefening starten. Je zult onderweg vragen beantwoorden, en je kan je voortgang volgen op de pagina Mijn Overzicht.
