Om te beginnen, moet je het HTML-bestand laten weten dat je JavaScript wilt gebruiken. Dit doe je op twee manieren. Of je zegt het volgende in je bestand: 

<script>Hier kan je de kleine JavaScript in stoppen die direct op de website uitgevoerd kan worden. Voor kleine scriptjes is dit een handige manier, maar voor grotere bestanden zoals
voor een chatbot is het handiger een extern bestand te gebruiken. Dit is manier twee.</script>
Ook kan je dit in de head of body zetten: 
<script src="externscript.js"></script>



Als je een script hebt ingeladen, kan je aan dit script werken om te zorgen dat het iets doet. Hier is een voorbeeld van een simpele JavaScript die een melding geeft in de browser:

<script>
      alert("Hello World!");
</script>


Hier kan je iets in de console van de browser laten zien. Dit heeft niets met de site zelf te maken, maar kan voor ontwikkelaars handig zijn om te vertellen waar de error in de code zit.

<script>
    console.log(error);
</script>


Ook kan je commentaar in je JS zetten, zodat andere mensen en jijzelf makkelijk kunnen zien wat een stukje code doet. Dit doe je op deze manier:

// Dit is een commentaar in JavaScript
