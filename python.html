Korte samenvatting over Python:

Python is een programmeertaal die is ontwikkeld door Guido van Rossum, een Nederlander! Het wordt veel gebruikt om programma's, spellen en AI-toepassingen te maken. Wat Python bijzonder maakt, is dat de code eruitziet als gewone zinnen, waardoor het makkelijk is om te begrijpen. Je kunt er bijvoorbeeld berekeningen mee maken, teksten veranderen of zelfs kleine spelletjes bouwen. Ook zijn veel AI-tools gemaakt met Python, omdat je er logische verbanden mee kan leggen en de computer gegevens aan elkaar kan laten koppelen. Je kan er ook programma's voor Windows mee maken. In deze cursus leer je de beginselen van deze taal, en met oefeningen breng je deze in de praktijk.
