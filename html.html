Korte samenvatting over Html:

HTML staat voor HyperText Markup Language, de taal die wij gebruiken om websites te maken. Het is de basis van elke webpagina die je ziet. HTML helpt ons om dingen op een website te plaatsen, zoals tekst, afbeeldingen en knoppen. Als je bijvoorbeeld een titel bovenaan je webpagina wilt hebben, gebruik je een speciaal stukje HTML-code namelijk H1 tussen pijltjes. Dit vertelt de browser dat het een grote titel is. Door HTML kun je dus de structuur van je website bepalen.


Als je er klaar voor bent, kan je met onderstaande knop de cursus starten. Je zult onderweg vragen beantwoorden, en je kan je voortgang volgen op de pagina Mijn Overzicht.
