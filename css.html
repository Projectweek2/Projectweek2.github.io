Korte samenvatting over CSS:

CSS staat voor Cascading Style Sheets en het word extreem vaak gebruikt om een website die gemaakt is in Html mooier te maken. Terwijl HTML de basis van je website maakt, zorgt CSS ervoor dat je kunt kiezen welke kleuren, lettertypes en vormen alles krijgt. Bijvoorbeeld, je kunt met CSS de tekst een andere kleur geven of de knoppen groter maken. Het is eigenlijk de manier om je website mooi aan te kleden, zodat alles er leuk en netjes uitziet.
